# Wesley Farias | Full Stack Portfolio

Portfólio profissional migrado para **Next.js + TypeScript** com foco em performance, SEO técnico e arquitetura escalável.

## Stack principal

- React.js
- Next.js (App Router)
- TypeScript
- Node.js
- NestJS (posicionamento técnico de backend)
- Tailwind CSS
- APIs REST
- SSR e SSG

## Objetivo do projeto

Apresentar serviços, experiência e projetos com posicionamento de Desenvolvedor Full Stack voltado para aplicações web modernas, performáticas, escaláveis e otimizadas para SEO.

## Estrutura

- `app/layout.tsx`: layout global e metadata SEO
- `app/page.tsx`: landing page principal com seções de projetos, serviços, experiência, sobre e stack
- `app/globals.css`: estilos globais e animação de marquee
- `public/images`: imagens dos projetos e seções

## Executar localmente

```bash
npm install
npm run dev
```

Aplicação disponível em `http://localhost:3000`.

## Build de produção

```bash
npm run build
npm run start
```

## Melhorias aplicadas na migração

- Migração da base frontend para Next.js App Router
- Atualização de conteúdo para Node.js, NestJS, React.js, Next.js e TypeScript
- Alinhamento completo do posicionamento técnico para Node.js, NestJS, React.js, Next.js e TypeScript
- Implementação de metadata SEO técnica (Open Graph e Twitter)
- Uso de `next/image` para otimização e lazy loading de imagens
- Melhorias de semântica HTML e acessibilidade (skip link, aria-label, foco visível)
